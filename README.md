# Crafting Interpreters - Traducción al Español

Este repositorio contiene la traducción al español del libro "Crafting Interpreters" escrita con la ayuda del servicio de traducción de [DeepL](https://www.deepl.com/). "Crafting Interpreters" es una guía completa para construir lenguajes de programación interpretados. 
**NOTA: La traducción del libro puede contener errores en el código proporcionado, recomiendo que veas el código de el libro original en inglés**

## Acerca del Libro

- **Título**: Crafting Interpreters
- **Autor**: Robert Nystrom
- **Sitio web oficial**: [craftinginterpreters.com](https://craftinginterpreters.com)
- **Versión original en inglés**: [repositorio en GitHub](https://github.com/munificent/craftinginterpreters)

El libro se divide en dos partes principales:

1. **Parte I: Un intérprete para un nuevo lenguaje de programación**. Esta parte proporciona una base sólida para construir lenguajes de programación interpretados. A medida que avances a través de los capítulos, aprenderás sobre la teoría y la implementación práctica de un intérprete.

2. **Parte II: Un intérprete para el lenguaje de programación de objetos Lox**. En esta parte, construirás un lenguaje de programación llamado Lox. Explorarás conceptos avanzados como el alcance de variables, la gestión de memoria, la evaluación de expresiones, la orientación a objetos y más.
